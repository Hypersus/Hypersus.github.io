Tips in linux bash terminal

Edit and Delete

  • ctrl+ u to delete contents left of the cursor
  • ctrl + k to delete contents right of the cursor
  • ctrl + w to delete a word left of the cursor

Navigate through the command line

  • ctrl + e to navigate to the end of the line
  • ctrl + a to navigate to the start of the line

Some wired thing

  • ctrl + s to lock the terminal (that is to deny any key stroke)
  • ctrl + q to unlock
